# Privacy Policy

**TimezonePeek** is designed with privacy in mind. The app does not collect, store, or transmit any personal data to us.

## Data Collection

- No personal information collected by the app
- No usage analytics or tracking
- No accounts or registration required

## External Services

TimezonePeek does not connect to any external services. The app works entirely offline.

- No network requests
- No API calls
- No third-party services

## Local Data

- Timezone list and labels are stored locally using UserDefaults
- Preferences (time format, display style, separator) are stored locally using UserDefaults
- All timezone calculations use macOS built-in TimeZone APIs
- No data is sent to us or any server

## Network Permission

TimezonePeek does not require network access. The app runs completely offline using your Mac's system timezone database.

## Sandbox

TimezonePeek runs in the macOS App Sandbox with minimal permissions:

- No network access
- No file system access
- No location services
- No camera or microphone
